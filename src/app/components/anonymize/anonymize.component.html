Ruls can only be used with JSON data
How it works

The `JSON path` is a dot-seperated string, which defines one or more fields to be modified. For example:

data.items.accounts.*.name
  • * will match any string
  • It will automatically traverse arrays
        
        {
          data: {
            items: [
              {
                 accounts: {
                   'o8hujhbhb': {
                      name: 'John', ..
                  },
                   '908huhkb': {
                      name: 'Bob', ..
                  },
                   ...
                }
              },
               ...
            ]
          }
        }
