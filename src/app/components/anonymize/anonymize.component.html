Rule TypeThis value describes what the field in question represents.

JSON Path is an expression which describes a field inside JSON. For example:

JSON path: $..accounts.*.name
  • $The root object/element
  • .The current object/element
  • ..Recursive descendant operator
  • *Wildcard matching all objects/elements regardless their names
  • A complete description of the syntax can be found  here
